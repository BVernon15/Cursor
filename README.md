## Push to GitHub (one-time auth)

From this folder in Terminal:

```bash
cd /Users/brycevernon/Projects/work-pages
git push -u origin main
```

When prompted, use your GitHub username and a **Personal Access Token** (not your password). Create one: GitHub → Settings → Developer settings → Personal access tokens → Generate new token (classic), scope `repo`.
